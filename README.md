
# Coin Collector

Steps to follow to run this project


1. install composer by running the commend 

```bash
  composer install
```

2. Generate the key 

```bash
  php artisan key:generate
```

3. Connect the Database

4. Run Migration

```bash
  php artisan migrate
```

4. Command to Collect Coin

```bash
  php artisan coin:collect
```
